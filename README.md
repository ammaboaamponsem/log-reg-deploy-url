# Income Prediction Microservice

## 📊 Overview
A machine learning microservice that predicts individual income brackets (>50K or ≤50K) using Census demographic data. This project demonstrates the full lifecycle of a predictive model: data cleaning, feature engineering, and deployment.

## 🛠 Tech Stack
- **Languages:** Python
- **Libraries:** Scikit-Learn, Pandas, NumPy
- **Deployment:** Flask/API Microservice

## 📈 Key Findings & Performance
- The model analyzes 14 features including education, occupation, and demographic factors.
- Achieved **80.28% predictive accuracy** on test datasets.
- Provides a scalable endpoint for real-time income classification.
